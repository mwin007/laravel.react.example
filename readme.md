## Laravel + React

Laravel + React + React-Router + Webpack

## 下載

git clone https://github.com/sexyoung/laravel.react.example

## 安裝 Laravel 套件
```
composer install
```
## 安裝 node 套件
```
npm install
```
## 啟動

```
php artisan serve --host=127.0.0.1
```
```
npm start
```

### 打開 瀏覽器 http://localhost:8080/
